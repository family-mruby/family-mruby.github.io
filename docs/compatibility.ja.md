## 接続のヒント

- **USB Hub 経由で複数同時接続** できます（キーボード + マウス + ゲームパッド）
- 一部の安価なマウスは **12bit packed プロトコル** を使い、起動直後の認識に再接続が必要なケースがあります（ホットプラグ対応済みなので、認識しないときは抜き差しを試してください）
- USB バスパワー機器の消費電流に注意。複数の電力大食い機器を同時接続する場合は **AC 給電付き USB Hub** を推奨
- USB ハブのチェーン段数が深いと低速デバイスの列挙に失敗することがあります

## 動作しない場合のチェック

1. デバイスが **USB HID 互換** か確認（独自プロトコルの製品は非対応）
2. 別の機器で動くか試す
3. USB Hub をはずして直接接続を試す
4. 給電を AC 駆動 Hub に切り替える
