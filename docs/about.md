# About

## About This Project

mkdoc_test is a template project for building documentation sites using MkDocs and Docker.

## Technologies Used

- **MkDocs**: Static site generator
- **Material for MkDocs**: Modern theme
- **Docker**: Build environment setup
- **GitHub Pages**: Site hosting
- **mkdocs-static-i18n**: Multilingual support

## Build Process

1. Build Docker image
2. Generate site with MkDocs
3. Deploy to GitHub Pages

## Multilingual Support

This site supports both Japanese and English. You can switch languages using the language selector in the navigation bar.
