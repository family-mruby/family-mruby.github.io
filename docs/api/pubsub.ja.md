# Pub/Sub

!!! info "工事中"
    このページは整備中です。

## 概要

アプリケーション間でメッセージをやり取りするための Publish/Subscribe 機構です。HID イベント（キーボード・マウス・ゲームパッド）の受信などに利用します。

## メソッド

| メソッド | 用途 |
|---|---|
| `subscribe(topic)` | トピック購読を開始 |
| `publish(topic, data)` | トピックに対してメッセージを送信 |
| `send_message(...)` | アプリ間メッセージ送信 |
