# About

## このプロジェクトについて

mkdoc_testは、MkDocsとDockerを使用してドキュメントサイトを構築するためのテンプレートプロジェクトです。

## 使用技術

- **MkDocs**: 静的サイトジェネレーター
- **Material for MkDocs**: モダンなテーマ
- **Docker**: ビルド環境の構築
- **GitHub Pages**: サイトのホスティング

## ビルド方法

1. Dockerイメージをビルド
2. MkDocsでサイトを生成
3. GitHub Pagesにデプロイ
